Чтобы посмотреть данные из БД mongodb нужно:
1) Открыть командную строку (запуск от администратора)
2) Прописать путь к bin
    cd Program Files\MongoDB\Server\4.2\bin
3) Далее последовательность команд:
    mongo
    show databases
    use dbsklad - (dbsklad - название БД, записано в config.js)
    show collections
4) Прописать необходимую команду
    db.products.fund({}) (показать все данные коллекции products)
    db.shippers.fund({}) (показать все данные коллекции shippers)
    db.products.drop() (удалить коллекцию products)
    db.shippers.drop() (удалить коллекцию shippers)
--------------------------------------------------------------------------------------------
Если удалили коллекцию, то чтобы добавить её в БД нужно раскомментить строчки в index.js

// Строчки для сохранения товаров из database.json в БД mongodb
//database.products.forEach(f => new Product(f).save().catch(e => console.log(e)))
//database.shippers.forEach(c => new Shipper(c).save().catch(e => console.log(e)))

Важно!
Как только проект запустится и бот начнёт работать, следует закомментить обратно эти строчки
Иначе в БД каждый раз будут создаваться дубликаты данных database.json
